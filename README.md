# 流行演唱个人展示网站

## 🎨 设计灵感

本网站模仿了 [Wire Music](https://wiremusic.sg/) 的现代化布局风格，采用以下设计特点：

### 布局特色

1. **大图横幅开场**
   - 使用全屏背景图片展示
   - 深色渐变遮罩层突出文字
   - 醒目的标题和副标题

2. **专业技能卡片展示**
   - 4个技能卡片网格布局
   - 每个卡片包含图片、图标和描述
   - 悬停时显示渐变遮罩和图标
   - 卡片上浮动画效果

3. **现代化成长历程时间线**
   - 左右交替的图文布局
   - 每个阶段配有标签徽章
   - 大图展示配合详细文字说明
   - 图片悬停放大效果

4. **作品展示区域**
   - 按类别分组展示照片
   - 网格布局，响应式设计
   - 点击照片全屏查看（灯箱效果）
   - 支持键盘左右切换

5. **视频作品展示**
   - 分类展示不同类型视频
   - 内置视频播放器
   - 自动暂停其他视频功能

6. **联系方式卡片**
   - 3个联系方式卡片
   - 圆形图标背景
   - 卡片悬停动画

### 配色方案

- **主题色**：紫色系 (#8B5CF6) - 代表创造力和艺术
- **辅助色**：粉色系 (#EC4899) - 代表热情和活力
- **点缀色**：金色系 (#F59E0B) - 代表成就和光芒
- **背景色**：深色系 (#0F0E17, #1A1625) - 现代感与专业感

### 交互特性

- ✨ 平滑滚动导航
- 🎯 悬停动画效果
- 🖼️ 照片灯箱查看
- ⌨️ 键盘导航支持
- 📱 完全响应式设计
- 🎬 智能视频播放控制

## 🚀 使用方法

1. 直接用浏览器打开 `index.html`
2. 修改联系方式（邮箱、微信）
3. 根据需要调整内容和照片分类

## 🎯 设计理念

采用音乐行业专业网站的布局风格，突出：
- 专业性：清晰的技能展示和成长历程
- 视觉冲击：大图展示和现代化动画

- 品牌感：统一的配色和视觉风格
