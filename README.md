# 流行演唱个人展示网站

## 🎨 设计灵感

本网站模仿了 [Wire Music](https://wiremusic.sg/) 的现代化布局风格，采用以下设计特点：

### 布局特色

1. **大图横幅开场**
   - 使用全屏背景图片展示
   - 深色渐变遮罩层突出文字
   - 醒目的标题和副标题

2. **专业技能卡片展示**
   - 4个技能卡片网格布局
   - 每个卡片包含图片、图标和描述
   - 悬停时显示渐变遮罩和图标
   - 卡片上浮动画效果

3. **现代化成长历程时间线**
   - 左右交替的图文布局
   - 每个阶段配有标签徽章
   - 大图展示配合详细文字说明
   - 图片悬停放大效果

4. **作品展示区域**
   - 按类别分组展示照片
   - 网格布局，响应式设计
   - 点击照片全屏查看（灯箱效果）
   - 支持键盘左右切换

5. **视频作品展示**
   - 分类展示不同类型视频
   - 内置视频播放器
   - 自动暂停其他视频功能

6. **联系方式卡片**
   - 3个联系方式卡片
   - 圆形图标背景
   - 卡片悬停动画

### 🎨 主题风格设计

#### 视觉主题：霓虹赛博朋克音乐风
- **整体基调**：深色背景 + 霓虹色彩
- **视觉特效**：发光效果、毛玻璃、渐变动画
- **设计理念**：现代、专业、富有科技感与艺术气息
- **灵感来源**：Wire Music 现代化音乐网站风格

#### 面板组件系统

##### 1️⃣ 导航栏面板（Navbar）
- **样式**：毛玻璃效果 + 渐变边框
- **特效**：`backdrop-filter: blur(20px)`
- **交互**：滚动时增加不透明度和发光效果
- **组件**：Logo（霓虹渐变文字）+ 导航链接（悬停发光）

##### 2️⃣ 个人资料卡片（Profile Card）
- **背景**：深色卡片背景 + 渐变装饰
- **布局**：头部标题 + 简介引用 + 技能图标
- **装饰**：学校徽章、技能统计面板
- **特效**：边框霓虹光晕

##### 3️⃣ 技能展示卡片（Skills Cards）
- **布局**：4列网格响应式布局
- **效果**：悬停时显示渐变遮罩层
- **组件**：图片背景 + 图标 + 标题 + 描述
- **动画**：上浮效果（`translateY(-10px)`）

##### 4️⃣ 时间线面板（Timeline）
- **样式**：左右交替布局
- **组件**：阶段标签 + 大图 + 详细描述
- **装饰**：时间线连接线、徽章标签
- **特效**：图片悬停放大、内容淡入动画

##### 5️⃣ 作品灯箱（Lightbox Modal）
- **结构**：全屏遮罩 + 居中内容
- **功能**：图片查看、左右切换、关闭
- **特效**：背景模糊、内容缩放动画
- **交互**：键盘导航（←/→/ESC）

##### 6️⃣ 视频播放面板
- **布局**：分类标题 + 视频网格
- **组件**：自定义播放器容器
- **功能**：智能暂停其他视频
- **特效**：悬停边框发光

##### 7️⃣ 联系方式卡片（Contact Cards）
- **布局**：3列网格
- **组件**：圆形图标背景 + 标题 + 详情
- **特效**：悬停上浮 + 阴影增强
- **样式**：霓虹渐变边框

## 🚀 使用方法

### 快速开始
1. **直接打开**：用现代浏览器打开 `index.html`
2. **本地服务器**（推荐）：
   ```bash
   # 使用 Python
   python -m http.server 8000
   
   # 使用 Node.js
   npx serve
   
   # 使用 VS Code Live Server 插件
   ```
3. **访问**：打开 `http://localhost:8000`

### 自定义内容

#### 修改个人信息
在 [index.html](index.html#L52-L55) 中修改：
```html
<h1 class="name-title">你的名字</h1>
<p class="major-text">你的专业 | 你的特长 | 所在地</p>
```

#### 修改联系方式
在 [index.html](index.html#L500-L530) 的联系区域：
- 邮箱地址
- 微信号
- Instagram账号

#### 调整配色
在 [style.css](style.css#L10-L20) 的 `:root` 变量中修改：
```css
:root {
  --neon-purple: #你的紫色;
  --neon-pink: #你的粉色;
  --neon-blue: #你的蓝色;
}
```

#### 更新照片和视频
1. 将新照片放入 `images/` 文件夹
2. 将新视频放入 `videos/` 文件夹
3. 在 HTML 中更新对应路径
